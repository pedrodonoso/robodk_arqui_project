
# Install virtualenv

```bash
    $ pip install virtualenv 
```

# Configurate virtualenv

```bash
    $ virtualenv -p /path/python venv
    $ source venv/bin/activate
    $ deactivate
```
# Install robodk

```bash
    $ pip install robodk
```
